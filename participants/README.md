# Registro de participantes

Cada participante debe crear **un archivo JSON propio** dentro de esta carpeta.

## Instrucciones

- Crea un archivo llamado exactamente `<tu-username-de-github>.json`
- El archivo debe incluir los campos `username`, `displayName` y `funFact`
- Abre un PR que modifique **solo ese archivo**
- No cambies otros archivos del repositorio

## ¿Por qué un archivo por persona?

Usamos un archivo por participante para evitar conflictos de merge cuando varias personas envian su PR al mismo tiempo.
